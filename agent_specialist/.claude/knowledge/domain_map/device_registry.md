## 成熟度说明

| 级别 | 特征 |
|------|------|
| L1 初级 | 依赖手册逐条查找；低置信输出多；需频繁人工校正 |
| L2 中级 | patterns/ 有积累；常见问题准确；偶尔需校正 |
| L3 高级 | 主动识别问题类型；自动套用框架；发现边界条件 |
| L4 专家 | 发现用户未提出的问题；系统级建议；输出可直接落地 |
